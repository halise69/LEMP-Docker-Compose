# LEMP Docker Compose

Linux Nginx Mysql Php & phpmyadmin

## Installation


```bash
git clone https://github.com/halise69/LEMP-Docker-Compose.git
```

## Start Docker

```bash
docker-compose up -d
```

## List Image Run Docker

```bash
docker ps -a
```

## Log Docker

```bash
docker-compose logs -f
```

## Remove Docker

```bash
docker-compose down
```
